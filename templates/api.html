Lincoln Park - Lakeview - North Pond Photo API

    api v1 (beta)

  • API Endpoints:
  • /api/v1/images/<image_name>
    Returns one image by image name a json object
  • /api/v1/images?<querystring>
    Returns one image with image name in query string as the actual image
  • /api/v1/images/all/<n>
    Returns array of n images as a json objects
  • /api/v1/images/<subject>/<n>
    Returns array of n images of <subject> as a json objects

Usage

  • To get a single image with the image name
    - <base_url>/api/v1/images/<image_name>

    Example: http://<base_url>/api/v1/images/0cec16c3-3146-4a8d-b4a1-5bc7065a817d.jpg

    - <base_url>/api/v1/images?image_name=<image_name>

    Example: http://<base_url>/api/v1/images?image_name=0cec16c3-3146-4a8d-b4a1-5bc7065a817d.jpg

  • To get <n> random images...
    - <base_url>/api/v1/images/all/<n>

    Example: http://<base_url>/api/v1/images/all/10

  • To get <n> random images about <subject>
    - <base_url>/api/v1/images/<subject>/<n>

    Example: http://<base_url>/api/v1/images/goose/10

  • Or you could try a custom search here
